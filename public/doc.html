Endpoint API

Endpoint utama untuk berinteraksi dengan AI.

  • URL: /api/chat
  • Method: POST
  • Content-Type: application/json

Body Request

{
  "query": "Apa itu JavaScript?",
  "customPrompt": "Jelaskan dengan gaya seorang bajak laut",
  "language": "Indonesia",
  "model": "default"
}
  • query (wajib): Teks pertanyaan Anda.
  • customPrompt (opsional): Sifat/karakter untuk AI.
  • language (opsional): Bahasa output yang diinginkan.
  • model (opsional): Pilihan model (`default`, `think-deeper`, `gpt-5`).
